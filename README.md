# Python Quiz Game

## Description

This is a console-based Quiz Game developed using Python.

## Features

* Multiple-choice questions
* User input validation
* Score tracking
* Displays correct and incorrect answers

## Technologies Used

* Python

## Concepts Used

* Functions
* Loops
* Conditional Statements
* Lists
* Dictionaries

## How to Run

1. Open the project folder in VS Code.
2. Open the terminal.
3. Run:

python quiz_game.py
